Assumptions

- all
  - support Promises
- browser
  - support `atob`, `btoa`
  - support `crypto.subtle.digest`
  - support `crypto.getRandomValues`
  - support `URL`
  - support `fetch`
  - support `EventSource`
  - support `Number.MIN_SAGE_INTEGER`, `Number.MAX_SAFE_INTEGER`, `isNaN`, `isFinite`
- non browser
  - support `Buffer`
  - support built in module `url`
